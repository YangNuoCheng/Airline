这是2020年的一个新的项目，思路如下：
1.通过网络爬虫获取中国（或全球）各地机场代码，机场之间的航班信息到数据库（后端）（Python + R + Mongodb）
2.对数据进行优化清洗，对应全球机场
3.使用html展示地图信息，并标注各个机场和航线（h5，d3)
4.可通过鼠标拖动方式查看
5.通过知识图谱形成智能问答

开发日志：
2020年3月24日，尝试从flightradar24官网爬取机场数据，但是爬取内容总为空，经查证巨大的网网页采用AJAX异步加载，request获取的只是第一部分的HTML源码，需要模拟发送AJAX请求，才能不断返回新的内容，扩充网页。CSDN：10944819
2020年3月25日，使用AJAX之后发现依然无法请求全部的内容。尝试保存完整的html，以后再查这个问题吧。
异步加载是个很难的问题，可以使用PhantomJS渲染页面（但是他已经停止更新，各大浏览的无头版本可以使用)，我最终使用selenium+Chrome的无头模式获得了网页的源代码）
2020年3月25日，使用以上方法顺利获得全球机场和航班数据。准备用HTML呈现.(全球数据会导出新建库)
2020年3月26日，将Mongodb中的数据清洗之后发往neo4j,获得更好的知识图谱体验。
